# 📊 IT Ticket Dashboard (Power BI)

## 📌 Project Description
This project is a Power BI dashboard developed to analyze IT support ticket data.  
It provides insights into ticket status, resolution time, technician performance, and ticket categories.

---

## ⚙️ Features
- 🎫 Ticket status tracking
- ⏱️ Resolution time analysis
- 👨‍💻 Technician performance monitoring
- 📈 Ticket trend analysis
- 📊 Interactive Power BI dashboard

---

## 🧑‍💻 Technologies Used
- Power BI
- Excel / CSV Dataset
- DAX
- Data Visualization
